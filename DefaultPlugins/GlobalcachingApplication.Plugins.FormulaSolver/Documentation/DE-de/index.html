Formula Solver Plugin

>>

Das Formula Solver Plugin ist ein Hilfsmittel zum Lösen cacherelevanter Aufgaben. Vor allem bei Mystery Caches kann es sehr von Nutzen sein. Es wird aufgerufen durch den Menüeintrag "Fenster\Formellöser" oder das entsprechende Icon in der Toolbar.

Zu jedem Cache der Cacheliste kann ein Satz Formeln gespeichert werden. Das Eingabefenster des Formellösers zeigt die zum markierten Cache gehörenden Einträge. Weiterhin stellt der Formellöser einen Eintrag bereit, der mit keinem Cache verknüpft ist und damit als Bereich für schnelle Berechnungen ohne Cachebezug genutzt werden kann. Dieser Eintrag ist immer dann zu sehen, wenn in der Cacheliste kein Cache ausgewählt wurde.
